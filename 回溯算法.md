# 回溯算法

解决一个回溯问题，实际上就是遍历一棵决策树的过程，树的每个叶子结点存放着一个合法答案。你把整棵树遍历一遍，把叶子结点上的答案都收集起来，就能得到所有的合法答案。

* 全排列问题
* N皇后问题

### 站在回溯树的一个节点上，核心思考点
1. 路径：也就是已经做出的选择。
2. 选择列表：也就是你当前可以做的选择。
3. 结束条件：也就是到达决策树底层，无法再做选择的条件。

### Demo
```
function backtrack(...) {
    if (到达叶子节点) {
        return;
    }

    for (let i = 0; i < n; i++) {
        // 做选择
        ...
        backtrack(...);
        // 撤销选择
        ...
    }
}
```
